# Create Video

Guidelines for producing video content across platforms.

## Tools

| Tool | Role |
|------|------|
| Claude Design | Script visualization, motion graphics, animated explainers |
| Screen Studio | Product demos and screen recordings |
| Canva | Thumbnails and intro/outro cards |

## Format guide

| Platform | Max length | Aspect ratio | Style |
|----------|-----------|-------------|-------|
| YouTube | 10-15 min | 16:9 | Tutorial, deep-dive, demo |
| LinkedIn | 1-3 min | 16:9 or 1:1 | Quick tip, product update |
| X/Twitter | 30-60 sec | 16:9 or 1:1 | Hook + one insight |
| Website | 2-5 min | 16:9 | Product demo, explainer |
| Loom (internal) | 5-10 min | 16:9 | Bug report, feature walkthrough |

## Process

1. **Script first** - write the script before recording; keep it tight and direct
2. **Hook in first 5 seconds** - state the problem or payoff immediately
3. **Create visuals:**
   - **Animated explainers / motion graphics**: use Claude Design to generate visual sequences, transitions, and animated diagrams
   - **Product demos**: use Screen Studio for screen recordings
   - **Thumbnails and cards**: use Canva brand kit
4. **Brand overlay** - company logo watermark (bottom-right, subtle), intro/outro cards
5. **Captions** - always add captions; most viewers watch on mute
6. **Export** - save to `content/youtube/_drafts/` or the relevant channel folder

## Recording checklist

- Clean desktop, no personal tabs visible
- Demo data only, no real customer data
- Good audio (external mic preferred)
- Consistent lighting if face is on camera

## Guardrails

- **No customer data** in screen recordings - use demo accounts only
- **Script review** before recording - don't wing it
- **Captions required** for all public videos
- **Thumbnail required** - no auto-generated frames
- **Save raw footage** alongside the export
