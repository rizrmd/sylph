# Create Presentation

Process for creating slide decks using the nao brand template.

## Process

1. **Load brand guidelines** - read `brand/brand-guidelines.md`
2. **Start from template** - always use Canva template `DAHGYLWJLOM`
3. **Outline first** - draft slide titles and key points before designing
4. **One idea per slide** - no walls of text; use visuals and short phrases
5. **Apply brand** - nao colors, Inter font, logo on title and closing slides only
6. **Export** - PDF for sharing, PPTX if editable version needed
7. **Save** - to `content/presentations/_drafts/YYYY-MM-DD_slug.pdf`

## Slide structure

- **Title slide**: presentation title, subtitle, date, nao logo
- **Agenda**: 3-5 items max
- **Content slides**: one point per slide, supporting visual or data
- **Closing slide**: CTA or next steps, contact info, nao logo

## Guardrails

- **Always use the Canva template** - never build from a blank deck
- **Max 15 slides** for external presentations; max 10 for internal
- **No clip art** - use screenshots, data charts, or simple icons
- **Run Hemingway rules** on all slide text before finalizing
