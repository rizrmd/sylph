# pnv

**A browser as a service.**

Record what you do in a browser. Turn it into a task. Replay it anytime.

No coding. No AI. Just a transcript of your interactions that you can run again.

---

## How it works

1. Open a cloud browser. Do what you normally do. Every click and type is recorded.
2. Select the steps you want. Inputs become variables automatically.
3. Run the task. Change the variables each time. Same flow, different data.

---

## Why it's different

| Problem | Instead of |
|---------|-----------|
| Writing and maintaining Playwright scripts | Just use a browser. Select steps from a transcript. |
| Prompting an LLM and hoping it clicks the right thing | Exact replay. Deterministic. Every time. |
| Doing the same data entry across accounts | Record once. Reuse with different values. |
