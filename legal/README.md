# Legal

Contracts, agreements, and legal templates.

## Structure

```
legal/
  templates/              - contract templates
    enterprise-service-agreement.md  - standard enterprise contract template
  _drafts/                - contracts in progress
  _published/             - signed contracts archive
```

## Skills

- `/create-contract` - generate a new enterprise contract from a proposal or deal terms
- `/review-contract` - review an incoming contract against standard positions
